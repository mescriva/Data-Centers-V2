KIOSKO TÁCTIL OFFLINE - GUÍA RÁPIDA (WINDOWS)
================================================

Este archivo está pensado para abrirse con Bloc de notas (Notepad).

IMPORTANTE (FORMATO DE ENTREGA)
------------------------------
El proyecto se entrega como un archivo ZIP.
En Windows debes descomprimirlo primero para extraer todos los archivos:
- Clic derecho sobre el ZIP -> "Extraer todo..."
- Elegir carpeta de destino y confirmar extracción

No ejecutes el proyecto desde dentro del ZIP comprimido.
Primero extrae y después abre/ejecuta los archivos desde la carpeta descomprimida.


1) QUÉ NECESITA EL ORDENADOR (WINDOWS)
--------------------------------------
Para reproducir la versión compilada (carpeta dist):
- Windows 10 u 11
- Navegador actualizado (Microsoft Edge o Google Chrome)
- Permiso para abrir localhost en el firewall (si aplica)

Si además quieres recompilar/editar el proyecto:
- Node.js 18 o superior (incluye npm)
- Visual Studio Code (recomendado)


2) CÓMO EJECUTAR LA VERSIÓN COMPILADA (RECOMENDADO PARA USO FINAL)
------------------------------------------------------------------
Antes de ejecutar, asegúrate de que la carpeta ZIP ya está descomprimida.

Opción A (más simple):
- Abrir la carpeta dist (ya extraída del ZIP).
- Doble clic en uno de los lanzadores .bat, por ejemplo:
  - 01-ABRIR-DATA-CENTERS.bat
  - 02-ABRIR-SOLAR.bat
  - 03-ABRIR-STORAGE.bat
  - 04-ABRIR-MOBILITY.bat

Opción B (manual con servidor local):
1. Abrir PowerShell en la carpeta dist.
2. Ejecutar:
   python -m http.server 8000
3. Abrir en navegador:
   http://localhost:8000

NOTA:
- No se recomienda abrir index.html directamente con file://
- Para funcionamiento correcto (Service Worker y almacenamiento), usar localhost.

IMPORTANTE PARA ACTUALIZACIONES:
- Si recibes una versión nueva, reemplaza la carpeta dist completa por la nueva.
- Los lanzadores .bat actualizados cierran automáticamente cualquier servidor anterior en el mismo puerto antes de abrir el kiosko.
- Si el navegador muestra versión antigua, cerrar todas las ventanas y volver a abrir el .bat.


3) CÓMO INSTALAR EL PROYECTO COMPLETO (SOLO SI VAS A DESARROLLAR)
------------------------------------------------------------------
1. Recibir/descargar el ZIP del proyecto.
2. Descomprimir el ZIP con "Extraer todo..." en Windows.
3. Abrir terminal en la raíz del proyecto descomprimido (no en dist).
4. Instalar Node.js LTS desde: https://nodejs.org/
5. Instalar dependencias:
   npm install
6. Levantar entorno de desarrollo:
   npm run dev
7. Compilar para producción:
   npm run build

La compilación vuelve a generar la carpeta dist.


4) CÓMO SE DESCARGAN LOS DATOS
------------------------------
Los eventos de uso se guardan en el navegador (IndexedDB/localStorage).

Para descargar los datos a Excel:
- Atajo de teclado: Ctrl + Shift + E
  o
- Abrir la ruta de administración en el navegador:
  http://localhost:5173/admin/export
  (si estás en otro puerto, cambiar el puerto)

En la ruta /admin/export:
- Pulsa el botón "Descargar Excel".
- Espera el mensaje de confirmación y revisa la carpeta Descargas.

Resultado:
- Se descarga un archivo .xlsx con nombre tipo:
  kiosk-interactions-YYYYMMDD-HHMM.xlsx

Recomendación:
- Ejecutar siempre en ventana normal (no incógnito) para asegurar guardado de datos.


5) PROBLEMAS FRECUENTES
-----------------------
- No abre el kiosko:
  Revisar que el navegador permita abrir localhost y que no bloquee scripts.

- No se guardan datos:
  Evitar modo incógnito y comprobar permisos de almacenamiento del navegador.

- No descarga Excel:
  Revisar bloqueadores de descarga del navegador y permisos de carpeta Descargas.
  Si entraste a /admin/export, confirmar que se pulsó el botón "Descargar Excel".


Fin de la guía.
